Beckhoff EP43xx 2Ch Analog Input/Output Slave

This block implements the Beckhoff's EtherCAT EP43xx series of Analog Input and Output fieldbus module.

The module has 2 configurable analog input ports and 2 configurable analog output ports. Choose the correct configuration using the provided selection lists.
